# Interactive Resume

A minimal, modern static resume built with Next.js, Tailwind CSS, and Framer Motion. 

## Features

- **Dark Theme:** Clean, high-contrast dark mode design.
- **Subtle Animations:** Smooth scroll and reveal effects via Framer Motion.
- **Interactive UI:** Features a dynamic grid background, spotlight highlight borders on cards, and an infinite scrolling skills carousel.
- **Static First:** No backend or database required.

## Managing Data

All resume content (Experience, Education, Skills, Projects, Certifications, and Blogs) is managed locally.
To update your data, simply edit: `data/profile.ts`.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Deployment

To build the static optimized version for production:
```bash
npm run build
npm run start
```
